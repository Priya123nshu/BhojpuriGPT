Model Information

Name: Bhojpuri GPT

Architecture: GPT-2 Large + LoRA

Parameters: 774M (Base) + 16M (Adapter)

Training Type: Causal Language Modeling / Finetuning

Dataset: conditioned_train.jsonl (Bhojpuri lyrics dataset)

Objective: Next-token prediction

Capabilities: Generating Bhojpuri lyrics, poems, passages, and regional dialogues.

Limitations: May hallucinate facts, limited context window (1024 tokens).

Training Details

Dataset Format: Plain text / formatted lyrics

Method: Parameter-Efficient Fine-Tuning (PEFT/LoRA)

Behavior: Tuned for generating continuous Bhojpuri text matching local flavor and rhythm.

Inference: Optimized for generation with Hugging Face transformers/peft.
